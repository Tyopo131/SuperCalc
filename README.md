# Welcome to SuperCalc
This calculator is like no other.  
You won't see some boring gui window with a on-screen keyboard, you're on a computer for god's sake, why need an on-screen keyboard?  
This calculator will open a **text editor** for the easy, natural experience of typing, and once you close the window  
C++ will blazingly fast parse your calculation and spit out a result in the same file as if by magic!
